# StrFormat,LCase

Converts a string to lowercase.

## Syntax

```pebakery
StrFormat,LCase,<String>,<%DestVar%>
```

### Arguments

| Argument | Description |
| --- | --- |
| String | The string to read. |
| DestVar | The variable where the result will be saved. |

## Remarks

None.

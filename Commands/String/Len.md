# StrFormat,Len

Returns the number of characters in a string.

## Syntax

```pebakery
StrFormat,Len,<String>,<%DestVar%>
```

### Arguments

| Argument | Description |
| --- | --- |
| String | The string to read. |
| DestVar | The variable where the result will be saved. |

## Remarks

None.

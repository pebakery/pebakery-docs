# StrFormat,UCase

Converts a string to uppercase.

## Syntax

```pebakery
StrFormat,UCase,<String>,<%DestVar%>
```

### Arguments

| Argument | Description |
| --- | --- |
| String | The string to read. |
| DestVar | The variable where the result will be saved. |

## Remarks

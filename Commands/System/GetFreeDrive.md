# System,GetFreeDrive

Returns the highest available free drive letter, Usually the "Z:" drive.

## Syntax

```pebakery
System,GetFreeDrive,<%DestVar%>
```

### Arguments

| Argument | Description |
| --- | --- |
| DestVar | The variable where the drive letter will be stored. |
